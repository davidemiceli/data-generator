Data Generation Platform for Statistical Learning and Machine Learning.

## Requirements

- Node.js v18+

## Getting started

The first time, build the frontend application (it requires Node.js and npm installed):
```bash
npm install
npm run build
npm run start
```

To run the development server:
```bash
npm run dev
```
